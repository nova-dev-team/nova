Nova/Core

This is the root page for "Core" component. You could use the following controls:

User: add new user, show user information, add/remove a vcluster from user

Vcluster: create new vcluster, delete vcluster, add/remove a vmachine from vcluster

Vmachine: create new vmachine, start/stop/suspend/resume vmachine

Pmachine: add new pmachine, mark/unmark a pmachine as "pending remove"

This page belongs to the "Core" component of Nova project, hosted at http://www.github.com/santazhang/nova.
